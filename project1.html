Chat and query bot using Document Retrieval, Indexing, and Generative models

  • by:Neelima Bollempalli

This project involved the development of a sophisticated chatbot capable of handling both casual conversation and factual queries by integrating document retrieval techniques with generative language models. The system was designed to interpret natural language input, retrieve relevant documents, and generate appropriate responses in real time.

Data Acquisition and Indexing

Information was first gathered using the Wikipedia Python library, which enabled efficient scraping of topic-specific content. The collected data was then indexed using Apache Solr through the Schema API, with PySol facilitating seamless interaction and storage. An inverted index structure based on the Document-at-a-Time (DAAT) approach was implemented to optimize Boolean query evaluation and ensure precise document matching.

Query Classification and Processing

A key component of the system was the ability to distinguish between chitchat and information-seeking queries. This classification enabled tailored handling, where casual inputs were answered using a dedicated conversational module, while factual questions were routed through a Solr-backed retrieval pipeline.

Interface and Enchancemets

The chatbot's backend was developed using Flask, which also handled tasks such as query parsing, spelling correction, sentiment analysis, and topic classification. The responses were further enhanced by incorporating BERT-based reranking, ensuring that the most semantically relevant information was prioritized in the final output.

The front-end interface was built using React, offering a clean and interactive experience for users to submit queries and receive real-time responses. Additional layers such as sentiment tracking using VADER and profanity filtering contributed to the overall robustness and user-friendliness of the system.

By combining traditional IR principles with deep learning techniques, this chatbot demonstrates a powerful and flexible approach to conversational AI. It has potential applications in customer support, education, healthcare information systems, and enterprise knowledge retrieval.
