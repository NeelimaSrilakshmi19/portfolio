Emotion recognition with Deep Learning

  • by:Neelima Bollempalli

Overview

The Emotion Recognition with Deep Learning project aimed to develop an intelligent system capable of identifying human emotions from facial expressions using advanced neural network architectures. By leveraging the FER-2013 dataset, the project adopted a data-driven approach to train deep learning models from scratch, focusing on accurately classifying emotions such as happiness, sadness, anger, and surprise. This initiative combined theoretical understanding with hands-on implementation, resulting in a real-time emotion detection pipeline powered by custom-built CNNs.

Dataset

The FER-2013 (Facial Expression Recognition 2013) dataset, a widely used benchmark in emotion detection research, was employed for model training and evaluation. It consists of over 35,000 labeled grayscale images of 48x48 pixels, each tagged with one of the seven emotional categories. The dataset ensured a data-driven and objective foundation for training the models.

Model Development

To explore the performance of different architectural choices, the various versions of the following custom neural networks were developed and trained from scratch:
  • ResNet: Introduced skip connections to ease the training of deeper networks by mitigating the vanishing gradient problem
  • MobileNet: A lightweight CNN optimized for mobile and embedded vision applications, using depthwise separable convolutions.
  • EfficientNet: A scalable model family that balances network depth, width, and resolution, offering high accuracy with fewer parameters.
Each model was trained using data augmentation techniques to improve generalization and prevent overfitting.

Implementation & Integration

Beyond training, the project extended to real-world application by integrating the trained models with a live webcam interface. This allowed the system to:
  • Detect faces in real time using OpenCV.
  • Preprocess and classify facial expressions into emotion categories.
  • Overlay emotion predictions on video feeds dynamically.
The final trained models achieved over 90% accuracy on the FER-2013 dataset, demonstrating their effectiveness in classifying facial emotions.
