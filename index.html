My Portfolio

RAG Chatbot

Retrieval-Augmented Generation (RAG) is a technique that improves language model responses by first retrieving relevant information from external sources containing real information and then using that data to generate accurate, grounded answers as opposed to a simple generative model which can only guess at the answer on basis of its limited training data.

Emotion recognition with Deep Learning

This project focuses on building a robust system that can detect human emotions from facial expressions using deep learning. Leveraging modern convolutional neural networks (CNNs), the system was designed to recognize seven basic emotions—such as happiness, sadness, anger, surprise, fear, disgust, and neutrality—from static images and real-time webcam feeds.

Credit Card Application Analysis

This project uses historical credit card application data to build a predictive model for credit approval decisions. We processed over 500,000 records, conducted exploratory data analysis, and engineered key features like employment status and income trends. The work supports automation of risk evaluation in the financial sector, improving both accuracy and speed of decisions.
